Connexion SSH vers un serveur Debian 12

1. Installer le serveur SSH (sur Debian 12)

su

Entrer le mot de passe root.

2. Installer le paquet SSH

apt update
apt install openssh-server

3. Vérifier que le service SSH fonctionne

systemctl status ssh

Si le service n’est pas actif :

systemctl start ssh
systemctl enable ssh

4. Connaître l’adresse IP du serveur

ip a

Repérer l’adresse IP sur l’interface réseau (ex. 192.168.1.50).

5. Vérifier le pare-feu (optionnel)

ufw status
ufw allow ssh
ufw reload

6. Connexion depuis un autre PC

Depuis Linux ou macOS :

ssh mustapha@192.168.1.50

Depuis Windows (PowerShell) :

ssh mustapha@192.168.1.50

7. Première connexion

Si le message suivant apparaît :

The authenticity of host '192.168.1.50' can't be established.
Are you sure you want to continue connecting (yes/no)?

Répondre yes et entrer le mot de passe de l’utilisateur.

8. Vérification

whoami
hostname

Résultat attendu :

mustapha
debian12

9. Quitter la session SSH

exit

Bonnes pratiques

  • Ne jamais se connecter directement en root via SSH
  • Utiliser un utilisateur avec sudo
  • Port SSH par défaut : 22
Astuce TSSR :
Pour plus de sécurité, désactiver le login root via SSH et utiliser des clés SSH.
