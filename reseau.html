Réseau - Révisions TSSR

Commandes IP de base

# Afficher la configuration IP
ip a

# Ajouter une IP à une interface
sudo ip addr add 192.168.1.10/24 dev eth0

# Supprimer une IP
sudo ip addr del 192.168.1.10/24 dev eth0

# Vérifier la table ARP
arp -a

# Vérifier la connectivité
ping 192.168.1.1

# Tracer la route vers une destination
traceroute google.com

Routage et VLAN (concepts)

  • Création de VLAN sur switch (exemple Cisco) : switchport access vlan 10
  • Encapsulation dot1q : encapsulation dot1q 20
  • Routage inter-VLAN via un routeur ou L3 switch

Services réseau

# Redémarrer le service réseau
sudo systemctl restart networking

systemctl status networking
