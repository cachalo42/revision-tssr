Installer sudo sur Debian (procédure correcte)

1. Passer en root

su

Entrer le mot de passe root.

2. Installer sudo

apt install sudo

3. Ajouter l’utilisateur au groupe sudo

/usr/sbin/usermod -a -G sudo mustapha

⚠️ Ne pas oublier l’option -a (append).

4. Quitter la session root

exit

5. Quitter la session utilisateur

exit

Cette étape permet de recharger les groupes de l’utilisateur.

6. Se reconnecter avec l’utilisateur mustapha

7. Vérifier l’appartenance au groupe sudo

groups

Si sudo apparaît dans la liste, l’installation est correcte.

8. Test final (optionnel)

sudo whoami

Résultat attendu :

root

Conclusion : L’utilisateur peut maintenant utiliser sudo pour exécuter des commandes administrateur.
