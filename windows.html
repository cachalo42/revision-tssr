Windows - Révisions TSSR

Gestion des utilisateurs (CMD)

# Créer un utilisateur
net user dupont MonMotDePasse123 /add

# Supprimer un utilisateur
net user dupont /del

# Ajouter à un groupe
net localgroup "Administrateurs" dupont /add

Active Directory (PowerShell)

# Créer un utilisateur
New-ADUser -Name "dupont" -AccountPassword (Read-Host -AsSecureString "Password") -Enabled $true

# Désactiver un utilisateur
Disable-ADAccount -Identity "durand"

# Ajouter à un groupe
Add-ADGroupMember -Identity "IT" -Members "dupont"

Gestion des dossiers et droits

# Créer un dossier partagé
mkdir D:\Partage
net share Donnees=D:\Partage /grant:IT,FULL

# Vérifier les droits NTFS
icacls D:\Partage

GPO (Group Policy)

  • Créer une GPO → gpmc.msc
  • Exemple : interdire le papier peint
  • Forcer l’application → gpupdate /force

Services Windows

# Vérifier un service
Get-Service -Name "Spooler"

# Redémarrer un service
Restart-Service -Name "Spooler"
