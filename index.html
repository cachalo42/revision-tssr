Linux - Révisions TSSR

Commandes essentielles

# Créer un utilisateur
sudo adduser dupont

# Modifier droits d’un fichier
chmod 644 fichier.txt
chown dupont:dupont fichier.txt

# Modifier sshd_config
sudo nano /etc/ssh/sshd_config
Port 2222
PermitRootLogin no
PasswordAuthentication no

Cas concret examen

  • Changer port SSH → 2222
  • Interdire root → PermitRootLogin no
  • Activer clés publiques → PasswordAuthentication no
  • Créer utilisateur test + droits
